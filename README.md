## :pushpin: 주요 기능

### 회원가입
- 유효성 검사

### 상품리스트 (기여도:100%)
- 이름 · 가격순 정렬

### 상품페이지 (기여도:100%)
- 옵션선택
- 수량선택
- Q&A · 리뷰 (Ajax)
- 바로구매 · 장바구니담기

### 장바구니
- 선택상품삭제

### 주문리스트
- 주문상세보기

### 관리자페이지 (기여도:100%)
- 상품관리 (등록, 삭제)
- 주문관리 (주문상세보기, 주문상태변경)

### 게시판 (기여도:100%)
- 공지사항
- Q&A

### 검색 (기여도:100%)
- 상품분류 · 상품명 · 가격으로 검색
- 이름 · 가격순 정렬
